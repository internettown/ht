## The game doesn't start/I don't know how to run it\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D1&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

### Windows\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D2&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

The problem might be caused by leftover data from older versions of Hardware Tycoon.

Navigate to your AppData folder:

C:\\Users\\<USERNAME>\\AppData\\Local

and delete the folder named HardwareTycoon.

WARNING: This action will also delete all saved games.

### macOS\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D3&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

##### "You do not have permission to open the application"\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D4&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

You have to add execution permissions to the application. You can do so using the terminal.

Navigate to the game's directory and launch the terminal there. Use either of these commands:

sudo chmod +x hardwaretycoon.app

sudo chmod 755 hardwaretycoon.app

##### The game refuses to start at all or throws an unhelpful error message\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D5&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

It's a common issue with nw.js applications. A workaround is to download the web version of the game using the itch client instead. [Here's](https://itch.io/post/604028) a step-by-step tutorial on how to do it.

### Linux\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D6&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

You have to add execution permissions to the application using this command:

chmod +x nw

in the game's directory.

Some distributions require you to launch it through the terminal, which you can do by running this command:

./nw

in the game's directory.

## My company's logo and all preset colors are red\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D7&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

This feature requires WebGL to be turned on.

### Browser version\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D8&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

Please refer to the official WebGL [website](https://get.webgl.org).

Things you can do:

-   Try a different browser,
-   Update your graphics drivers,
-   Try running the game using a different video card.

### Desktop versions\[[](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fhardware-tycoon.fandom.com%2Fwiki%2FTroubleshooting%3Fveaction%3Dedit%26section%3D9&uselang=en&metadata=article-registration-edit-article-section "Sign in to edit")\]

Please make sure that your system supports OpenGL.

Things you can do:

-   Update your graphics drivers,
-   Try running the game using a different video card.
